# Daglief PDP assets

Publieke assets voor de Daglief PDP bestelkeuze.

## Inladen

```html
<link rel="stylesheet" href="https://sparklesrob.github.io/daglief-pdp-assets/pdp-order-choice.css?v=1">
<script src="https://sparklesrob.github.io/daglief-pdp-assets/pdp-order-choice.js?v=1" defer></script>
```

Verhoog `?v=1` naar `?v=2`, `?v=3`, enzovoort na wijzigingen zodat browsers de nieuwste versie ophalen.

## Bestanden

- `pdp-order-choice.css`
- `pdp-order-choice.js`

`pdp.html` staat bewust in `.gitignore`, zodat alleen de losse assetbestanden gepubliceerd worden.
